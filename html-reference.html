HTML

Decsription

Html stands for HyperText Markup Language which is a standarized text system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages.

Syntax

start tag: <html>
end tag: </html>

Example

start tag: <html>
content: head and body elements
end tag: </html>

W3schools HTML Page

Head

Decsription

The Head represents a container for introductory content or a set of navigational links.

Syntax

start tag:<head>
end tag:</head>

Example

<head>Pancakes</head>

Title

Decsription

The title tag is required in all HTML documents and it defines the title of the document.

Syntax

start tag:<title>
end tag:</title>

Example

<title>repl.it</title>

Body

Decsription

The Body tag defines the main content of the HTML document or the section of the HTML document that will be directly visible on your web page.

Syntax

start tag:<body>
end tag:</body>

Example

</body>

Syntax

end tags:</h1></h2></h3>

Example

Paragraph

Decsription

Paragraphs usually represented in visual media as blocks of text that are separated from adjacent blocks by vertical blank space

Syntax

start tag:<p>
end tag:</p>

Example

Display

Comments

Decsription

HTML comments are visible to anyone that views the page source code, but are not rendered when the HTML document is rendered by a browser.

Syntax

<!-- -->

Example

Images

Decsription

Images are not technically inserted into an HTML page, images are linked to HTML pages. The tag creates a holding space for the referenced image.

Syntax

<img>

Example

<img src="img_girl.jpg" alt="Girl in a jacket">

Display

Lists

Decsription

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag. An ordered list starts with the <ol> tag.

Syntax

start tags:<ul>, <li>, <ol>
end tags:</ul>, </li>, </ol>

Example

  1. Coffee
  2. Tea
  3. Milk
W3schools Lists Page

Tables

Decsription

Each table row is defined with the <tr> tag. A table header is defined with the <th> tag. By default, table headings are bold and centered. A table data/cell is defined with the <td> tag.

Syntax

start tags:<tr>, <th>, <td>
end tags:</tr>, </th>, </td>

Example

<tr>
table elements
</tr>

Display

Month Savings
January $100
W3schools Tables Page

Blocks

Decsription

A block-level element always starts on a new line and takes up the full width available. The <div> element is a block-level element.

Syntax

start tag:<div>
end tag:</div>

Example

<div>
some text here
</div>
